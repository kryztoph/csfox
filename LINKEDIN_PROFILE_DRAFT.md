## 3. Experience Section (Enhancement Tips)
**Goal:** Focus on *Achievements*, not just *Responsibilities*.

For your role at **CGI** and previous positions, try to rewrite bullet points using the **STAR method** (Situation, Task, Action, Result).

*   **Bad:** "Responsible for writing Java code for financial reports."
*   **Good:** "Engineered a custom Java reporting engine that reduced monthly processing time by 40%, saving the finance team 20 hours per week."

**Key Verbs to Use:**
*   Automated
*   Optimized
*   Architected
*   Reduced (costs/time)
*   Increased (efficiency/revenue)
